# Sales Data Analysis Dashboard

## 📌 Overview
This project presents an interactive sales dashboard built using Excel/Power BI to analyze sales performance, trends, and key business insights.

## 🎯 Objectives
- Analyze sales trends over time
- Identify top-performing products and regions
- Track KPIs like total sales and profit
- Support data-driven decision making

## 📊 Dataset
- Sales data including product, region, date, revenue, and profit
- Dataset size: ~200+ rows

## 🛠 Tools Used
- Excel / Power BI
- Pivot Tables
- Data Visualization
- python

## 📈 Dashboard Features
- KPI Cards (Total Sales, Profit)
- Sales trend over time
- Region-wise performance
- Product category analysis
